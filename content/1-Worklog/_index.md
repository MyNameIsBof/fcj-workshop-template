**On this page**, you will need to introduce your worklog. **How** did you complete it? How many weeks did you take to complete the program? **What** did you do in those weeks?

Typically, and as a standard, a worklog is carried out over about 3 months (throughout the internship period) with weekly contents as follows:

**Week 1:** [Connect and get acquainted with members of First Cloud Journey and understand basic AWS services](1.1-week1/)

**Week 2:** [Master knowledge of network security in AWS (Security Groups, NACL) and load balancing with AWS ELB](1.2-week2/)

**Week 3:** [Get to know First Cloud Journey members and understand fundamental AWS services](1.3-week3/)

**Week 4:** [Master Amazon Simple Storage Service (S3) and its ecosystem](1.4-week4/)

**Week 5:** [Master the AWS Shared Responsibility Model and identity and access management on AWS](1.5-week5/)

**Week 6:** [Master core and advanced database concepts and gain proficiency in AWS database services](1.6-week6/)

**Week 7:** [Initiate Human Resource Management System project with Face Recognition Attendance](1.7-week7/)

**Week 8:** [Build frontend foundation with React 19 and Vite, implement routing structure and layout components](1.8-week8/)

**Week 9:** [Implement Employee Management module with CRUD operations and User Management system](1.9-week9/)

**Week 10:** [Implement Face Recognition system backend with Python Flask and face detection algorithms](1.10-week10/)

**Week 11:** [Integrate Attendance Tracking with face recognition system and develop Payroll Management module](1.11-week11/)

**Week 12:** [Develop Reports & Analytics module with data visualization and implement Document Management system](1.12-week12/)
